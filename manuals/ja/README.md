# ScriptVoxel ブロックゲーム - プレイヤーマニュアル

ScriptVoxel ブロックゲームへようこそ！これは、マルチプレイヤーサポートとユニークなPythonプログラマブルエージェントシステムを備えた、Minecraftライクなボクセルサンドボックスゲームです。

## クイックスタート

1. **ゲームを起動** - voxelモジュール付きのGodotを実行
2. **モードを選択** - シングルプレイヤー、サーバーに接続、またはサーバーをホスト
3. **プレイ開始** - WASDで移動、マウスで視点操作、クリックでブロックと相互作用

## クイックリファレンス

### 基本操作

| アクション | キー/ボタン |
|---------|-----------|
| 移動 | W/A/S/D または 矢印キー |
| ジャンプ | SPACE |
| 視点操作 | マウス移動 |
| ブロック配置 | 右クリック |
| ブロック削除 | 左クリック |
| ブロックピック | 中クリック |
| インベントリを開く | E |
| ホットバースロット選択 | 1-9 キー |
| マウス解放 | ESC |

### ゲームモード

- **シングルプレイヤー** - Pythonエージェントサポート付きで一人でプレイ
- **クライアント** - マルチプレイヤーサーバーに接続
- **ホスト** - 他のプレイヤーが参加できる独自のサーバーを実行

### 初期インベントリ

ゲーム開始時、ホットバーには以下のアイテムが入っています:
1. 土
2. 草
3. 石
4. 砂
5. 板材
6. 丸太
7. 葉
8. オーク板材
9. ロケットランチャー

## 特殊機能

### Pythonプログラマブルエージェント(シングルプレイヤーのみ)
**F4**を押してPythonコードエディタを開き、ゲームワールドと相互作用するAIエージェントをプログラムできます。これは自動化と実験のためのユニークな機能です。

### 自動セーブ
ゲームを閉じると(シングルプレイヤー)、またはサーバーをシャットダウンすると(マルチプレイヤーホスト)、ワールドが自動的に保存されます。

## このゲームについて

ScriptVoxel ブロックゲームは、Godot Voxelモジュールのデモ/テストプロジェクトです。サンドボックス環境でボクセル地形生成、マルチプレイヤー同期、インタラクティブなゲームプレイを紹介しています。

**注意:** このゲームは、コンパイルされたgodot_voxelモジュールを含むGodot 4.4が必要です。標準のGodotビルドでは実行できません。
