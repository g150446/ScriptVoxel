# ScriptVoxel Blocky Game - Player Manual

Welcome to the ScriptVoxel Blocky Game! This is a Minecraft-like voxel sandbox game with multiplayer support and a unique Python programmable agent system.

## Quick Start

1. **Launch the game** - Run Godot with the voxel module
2. **Choose a mode** - Singleplayer, connect to server, or host server
3. **Start playing** - Use WASD to move, mouse to look, click to interact with blocks

## Quick Reference

### Essential Controls

| Action | Key/Button |
|--------|-----------|
| Move | W/A/S/D or Arrow Keys |
| Jump | SPACE |
| Look Around | Move Mouse |
| Place Block | Right Click |
| Remove Block | Left Click |
| Pick Block | Middle Click |
| Open Inventory | E |
| Select Hotbar Slot | 1-9 Keys |
| Release Mouse | ESC |

### Game Modes

- **Singleplayer** - Play alone with Python agent support
- **Client** - Connect to a multiplayer server
- **Host** - Run your own server for others to join

### Starting Inventory

When you start the game, you begin with these items in your hotbar:
1. Dirt
2. Grass
3. Stone
4. Sand
5. Planks
6. Log
7. Leaves
8. Oak Planks
9. Rocket Launcher

## Special Features

### Python Programmable Agent (Singleplayer Only)
Press **F4** to open the Python code editor and program an AI agent to interact with the game world. This is a unique feature for automation and experimentation.

### Auto-Save
Your world automatically saves when you close the game (singleplayer) or shut down the server (multiplayer host).

## About This Game

ScriptVoxel Blocky Game is a demo/test project for the Godot Voxel module. It showcases voxel terrain generation, multiplayer synchronization, and interactive gameplay in a sandbox environment.

**Note:** This game requires Godot 4.4 with the compiled godot_voxel module. It cannot run on standard Godot builds.
